# **App Name**: PropertyLink

## Core Features:

- User Roles: Implement visitor, owner/agent, and admin roles with specific permissions.
- Property Posting: Enable owners/agents to post properties with details like photos, videos, price, size, and location.
- Property Search: Allow visitors to search properties with filters for location, price range, BHK, and furnishing.
- Property Details: Show detailed information for each property, including gallery, map location, owner contact, amenities, and similar properties.
- Favorite Properties: Allow users to save favorite properties for later viewing.
- Admin Approval: Require admin approval for all property listings before they are visible to the public.
- AI-Powered Property Matching: Use a generative AI tool to suggest similar properties based on user preferences and property features, enhancing property discovery.

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) to evoke trust and stability, reflecting the real estate domain.
- Background color: Light gray (#F0F4F7), creating a clean and professional look.
- Accent color: Warm orange (#E59866) for CTAs and highlights to draw attention.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern and neutral look.
- Use clear, consistent icons for property features, amenities, and filters.
- Clean and intuitive layout with a focus on property images and key information. Prioritize filter visibility on listing pages.
- Subtle transitions and loading animations to enhance user experience and provide feedback during interactions.
